## ğŸ¯ Algorithm

The generator uses a **greedy optimization** approach:

1. Place N pegs evenly around a circle
2. Start with a white canvas
3. For each peg, find the connection that reduces the most "error"
4. Draw that string (semi-transparent black line)
5. Repeat until convergence or limits reached

Each string darkens pixels along its path. By always choosing the line that most needs darkening, the image progressively emerges through overlapping threads.
